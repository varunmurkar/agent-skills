# Communication Standards

## Core Principles

- Keep transient plans, logs, and summaries in chat.
- Do not create unsolicited analysis files.
- Lead with key decisions and outcomes, then rationale.
- Highlight risks and blockers clearly.
- Do not use emoji for everything.

## Decision-Making

- Ask for user input only when:
  - authoritative sources conflict,
  - critical resources are inaccessible,
  - action risks irreversible damage,
  - ambiguity remains after reasonable investigation.
- Otherwise proceed autonomously with verifiable evidence.

## Status Reporting

- Use concise status markers: `✅` success, `⚠️` self-corrected issue, `🚧` blocker.
- Include next steps when follow-up is needed.
